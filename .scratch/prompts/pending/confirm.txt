## 分析规则

1. **条件分支必须完整**：每个 if-else、switch-case、三元表达式都要体现在流程图中
2. **异常必须标注**：每个 Asserts.check / throw 都要列出，标明触发条件
3. **事务边界必须清晰**：标注 @Transactional 的方法、异步执行的代码块、finally 中的操作

5. **外部系统调用必须标注**：HTTP 调用、MQ 发送、缓存操作

## Service 方法（第二层）
### ReconcileUsageApp.confirm
```java
    @Transactional(rollbackFor = Exception.class)
    @Override
    public List<ReconcileDetailJudgeAmountResp> confirm(ReconcileUsageConfirmReq confirmReq) {

        //1.根据对账单编码查询信息
        ReconcileUsageBatchBasicInfoDto basicInfoDto = purReconcileUsageBasicDomain.batchGetByReconcileCodes(confirmReq.getReconcileCodeList());
        if (ObjectUtils.isEmpty(basicInfoDto) || CollectionUtils.isEmpty(basicInfoDto.getReconcileInfos())) {
            return new ArrayList<>();
        }
        List<PurReconcileUsageEntity> reconcileInfoList = basicInfoDto.getReconcileInfos();
        List<PurReconcileUsageDetailsEntity> detailsEntities = basicInfoDto.getReconcileUsageLineDetailList();

        //4.按照头进行分组
        Map<String, List<PurReconcileUsageDetailsEntity>> reconcileCodeMap = ListUtils.emptyIfNull(detailsEntities).stream().collect(Collectors.groupingBy(PurReconcileUsageDetailsEntity::getReconcileCode));
        //5.走通用审批
        List<ReconcileUsageStrategyReq> reqList = ListUtils.emptyIfNull(reconcileInfoList).stream()
                .map(reconcile -> ReconcileUsageStrategyReq.builder()
                        .usageNodeActionEnum(UsageNodeActionEnum.PASS)
                        .submitReq(ReconcileUsageSubmitDto.builder().submitRemark(confirmReq.getConfirmRemark())
                                //确认账单按钮
                                .actionButton(ReconcileActionButtonEnum.CONFIRM_BILL.getCode())
                                .submitBatchKeys(confirmReq.getConfirmBatchKeys()).build())
                        .usageNodeSourceEnum(UsageNodeSourceEnum.RECONCILE_CONFIRM)
                        .localReconcileInfo(reconcile)
                        .localReconcileDetailInfo(reconcileCodeMap.get(reconcile.getReconcileCode()))
                        .build()
                ).collect(Collectors.toList());

        businessSourceNodeApp.batchApproveOperate(reqList);

        return new ArrayList<>();
    }
```
